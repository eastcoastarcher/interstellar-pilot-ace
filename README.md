# Interstellar Pilot2: A.C.E

## Archer's Custom Edition

This is edition of Interstealler Pilot 2 is a personl mod editio.

This mode is a bit more closer to the original stock game-play and feel, however with less micro-managing each ship detail for more fun game-play.

Some game play featuers:
- Updated graphics (same as contiuum)
- New sound track (same as continuum)
- Infinite Generators and capacitors
- Every shp has unique maxxed out engines and shielding
- Photon Bursts are the Anti-Fighter weapons, does direct-point shield damage
- Plasma Pulse is most powerful but slow, does half-splash shield damage
- Lasers do great agains shields but have a high miss rat, does full-splass shield damage
- Mines do more powerful damage
- Missiles are more powerful

Enjoy.
